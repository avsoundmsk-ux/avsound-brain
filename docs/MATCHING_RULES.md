# Назначение

Правила поиска товара на сайте и определения «уверенного» совпадения.

# Порядок поиска
1. Артикул
2. SKU
3. Модель
4. Полное название
5. Похожее название (fuzzy)

# Уверенное совпадение (можно обновлять)
Подтверждено одновременно:
- совпадает артикул ИЛИ SKU; ИЛИ
- совпадает (бренд + модель).

# Сомнительное (НЕ менять → ручная проверка)
- совпадает только название;
- несколько совпадений;
- fuzzy ниже порога.

# Проверка на дубликаты ПЕРЕД созданием
Перед созданием нового товара проверить:
- SKU;
- артикул;
- модель;
- бренд + модель;
- нормализованное название.

При сомнении — НЕ создавать, отправить в ручную проверку.
