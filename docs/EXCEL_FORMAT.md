# EXCEL_FORMAT.md

Описывает формат входного Excel: колонки, типы, обязательность, трактовка пустых значений.

# Структура файла (по факту `Копия склад (8).xlsx`)

- Лист: первый/активный.
- Строка 1 (индекс 0) — пустая.
- Строка 2 (индекс 1) — ЗАГОЛОВОК колонок.
- Далее идут блоки по брендам: строка-секция с одним значением = название бренда (напр. `Quart`, `JBL`, `Nakamichi`, `Ural`), затем строки товаров этого бренда.
- Пустые строки-разделители между блоками.

В файле `(8).xlsx`: 4 бренда, 167 строк-товаров.

# Колонки

| № | Заголовок | Поле | Тип | Обяз. | Описание | Пример |
|---|---|---|---|---|---|---|
| A | Статус | status | строка | да | статус объявления Avito | `Активно` |
| B | состояние | condition | строка | да | состояние товара | `Новое` |
| C | GoodsType | goods_type | строка | да | тип товара Avito | `Аудио- и видеотехника` |
| D | категория | category | строка | да | категория товара | `Автоакустика` |
| E | название | name | строка | **да** | полное название (бренд + модель) | `MB Quart M1-216` |
| F | стоимость | price | число/строка | да | цена продажи, ₽ | `5600` |
| G | количество | qty | число | да | остаток на складе | `2` |
| H | закуп | cost | число | нет | закупочная цена (может быть пусто) | `3195` |
| I | то что нет на авито | note | строка | нет | пометка | пусто |

# Бренд
- Отдельной колонки бренда НЕТ.
- Бренд определяется по строке-секции (заголовок блока) И/ИЛИ из начала названия (колонка E).

# Артикул / SKU
- Отдельной колонки артикула/SKU в файле НЕТ.
- Поиск и сопоставление товара — по названию/модели (см. MATCHING_RULES.md). Это ключевой риск: матчинг только по тексту.

# Правила трактовки
- Строка, где заполнена только колонка A (одно значение) и это известный бренд → секция, НЕ товар.
- Полностью пустая строка → пропуск.
- Нет названия (E пусто) → пропуск + в отчёт.
- Нет цены (F пусто) → товар в ручную проверку (цену не выдумывать).
- Нет закупа (H пусто) → допустимо, не блокирует.
- Категория (D) используется для назначения категории WooCommerce (см. CATEGORY_RULES.md).
