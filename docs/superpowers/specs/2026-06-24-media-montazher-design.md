# Дизайн: Агент «Монтажёр» (локальный медиа-режиссёр)

Дата: 2026-06-24

## Контекст

Михаил (AVSound) генерит видео/фото для маркетинга. Раньше — Higgsfield (дорого,
много брака). Перешли на KIE.ai (pay-per-use, дешевле). За последнюю сессию вручную
сделали: оживление фото усилителя (Seedance i2v) и face-swap карточку (Nano Banana).
Появились готовые хелперы `tools/kie_gen.py` и `tools/_run_face.py`, ключ KIE и правила
в памяти.

Проблема: каждый раз заново вспоминаем какую модель брать, как считать бюджет, как
писать промт. Нужно упаковать это в **систему** — чтобы Claude действовал как штатный
видео/фото-монтажёр: сам выбирает модель под задачу, пишет промт, генерит, отдаёт файл.

Решение принято: **локальный навык в этом Claude Code проекте** (НЕ в Telegram-боте
avtozvuk-agent, без деплоя). «Монтажёром» выступает Claude в чате.

## Цель

Сказал «сделай ролик/фото из этого» → Claude:
1. выбирает подходящую модель KIE (любую с сайта, не из зашитого списка),
2. пишет промт под неё,
3. проверяет бюджет, генерит через KIE API,
4. скачивает и отдаёт готовый файл + пишет в лог цену.

## Компоненты

### 1. Универсальный хелпер `tools/kie_gen.py` (расширить существующий)
- Подкоманда `generate`: принимает ЛЮБОЙ `--model <kie_model_id>` + `--prompt` +
  входные медиа (`--image`, `--images`, `--video`, `--videos`) + произвольные
  passthrough-параметры (`--set key=value`, напр. resolution, duration, aspect_ratio).
- Авто-загрузка локальных файлов в KIE (base64 → `https://kieai.redpandaai.co/api/file-base64-upload`),
  подстановка вернувшихся URL в нужные поля input (first_frame_url / image_urls /
  reference_video_urls — по типу модели/флагу).
- Опрос `recordInfo`, извлечение результата (видео .mp4 / картинка .png|jpg),
  авто-скачивание в `media/generated/` с браузерным User-Agent (иначе 403).
- Баланс до/после, запись в `media/generated/_log.md` (модель, входы, ~cr, ~$, файл).
- Оценка цены: для известных моделей (таблица RATES) — точная; для незнакомых —
  показываем баланс до/после, точную стоимость берём по факту списания.
- Сохранить обратную совместимость с текущими вызовами (`--image`, `--res`, `--dur`).
- `_run_face.py` остаётся как пример, но логику переносим в `generate`.

### 2. Каталог моделей `tools/kie_models.md` (сгенерировать скрейпом KIE)
- Стянуть список моделей с `https://kie.ai/market` (+ docs) firecrawl'ом.
- Для каждой: model_id, тип (image/video/audio), ключевые входы, цена (cr/сек или /шт),
  заметка когда брать. Кэш-файл, обновляемый по запросу.

### 3. Навык `.claude/skills/media-montazher/SKILL.md`
- Авто-триггер: «сделай/сгенерь видео|ролик|фото|картинку», «оживи фото», «face-swap»,
  «промт для видео», «какой моделью».
- Содержит:
  - **Матрицу задача→модель** (гид, не ограничение):
    | Задача | Модель (старт) | Почему |
    |---|---|---|
    | Перенести человека/лицо в сцену | nano-banana-edit (ТОЛЬКО 1 фото + текст) | держит лицо |
    | Оживить готовое фото | seedance i2v | дёшево, кадр верный |
    | Картинка с нуля | nano-banana | дёшево |
    | Динамика/сложное движение | kling | лучшее движение |
    | Реализм + нативный звук | veo 3 / seedance+audio | качество |
    | Длинный дешёвый ролик | seedance-2-fast 720p | цена |
  - Правило «бери любую модель с сайта» + ссылка на `tools/kie_models.md`.
  - Правила бюджета: KIE credit = $0.005; проверять баланс перед генерацией;
    i2v-видео (видео-референс) дешевле t2v; фото-старт у Seedance считается по полной ставке.
  - Шаблоны промтов под модели (face-swap, i2v-оживление, t2v-сцена).
  - Ссылки на правила в памяти: [[kie-nano-banana-faceswap]], [[api-keys]],
    [[media-generation-context]].
  - Как звать хелпер: примеры команд `python tools/kie_gen.py generate ...`.

## Поток

```
Михаил: "сделай ролик из этого фото" (+ файл/путь)
  → Claude (Монтажёр): определяет задачу → выбирает модель (матрица/каталог)
  → пишет промт → проверяет баланс → python tools/kie_gen.py generate --model ... 
  → хелпер: upload → createTask → poll → download → log
  → Claude отдаёт файл + цену + остаток
```

## Границы (YAGNI)
- НЕ трогаем Telegram-бот avtozvuk-agent, НЕ деплоим на Hetzner.
- НЕ строим UI. Запуск — через Claude в чате.
- Цену незнакомых моделей не угадываем — берём по факту (баланс до/после).

## Проверка
1. `python tools/kie_gen.py generate --model google/nano-banana-edit --image <локальное фото> --prompt "..."` → выдаёт PNG в media/generated/.
2. `python tools/kie_gen.py generate --model bytedance/seedance-2 --image <фото> --set resolution=720p --set duration=8 --audio --prompt "..."` → выдаёт MP4.
3. Каталог `tools/kie_models.md` существует и содержит ≥5 моделей с ID и ценами.
4. Skill активируется на фразу «сделай ролик из фото» (проверить что Claude сам зовёт хелпер).
5. Баланс KIE сейчас ~2 cr — для реального теста нужно пополнение; до пополнения проверяем сухой прогон (сборка payload, оценка цены) без списания.
