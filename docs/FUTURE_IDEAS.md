# FUTURE_IDEAS.md

# План развития проекта

Этот документ содержит все идеи, которые будут реализованы в будущих версиях проекта.

Новые идеи не должны сразу попадать в разработку.

Сначала они записываются сюда.

После анализа принимается решение о реализации.

---

# Версия 1.0

Основная задача:

Автоматическое создание и обновление карточек товаров AV-Sound.ru.

Функции:

* импорт Excel;
* поиск товаров;
* создание товаров;
* обновление товаров;
* поиск характеристик;
* создание описаний;
* поиск изображений;
* генерация через KIE.ai;
* загрузка изображений;
* отчеты.

---

# Версия 2.0

Интеллектуальный анализ товаров.

Добавить:

* автоматическое сравнение товаров;
* поиск аналогов;
* поиск устаревших моделей;
* автоматическую замену снятых с производства товаров;
* автоматическое определение поколения модели.

---

# Версия 3.0

Автоматическая работа с ценами.

Добавить:

* анализ цен конкурентов;

* рекомендованная цена;
* автоматическое изменение цен по правилам.

# AI-модели

Проект должен поддерживать возможность подключения любых моделей.

Например:

* KIE.ai;
* Nano Banana;
* Seedance;
* Gemini;
* GPT;
* Claude;
* Flux;
* Stable Diffusion;
* Midjourney;
* OpenRouter;
* другие API.

Замена одной модели не должна требовать переписывания проекта.

* количество запросов;

---

# Главная цель проекта

Постепенно превратить систему из простого обработчика Excel в полноценную AI-платформу управления каталогом интернет-магазина.

В будущем система должна уметь самостоятельно поддерживать каталог в актуальном состоянии с минимальным участием человека.
