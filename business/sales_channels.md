# каналы продаж AVSound

## 1. студия установки

источники клиентов:
- Яндекс Карты
- 2ГИС
- сарафанное радио
- Telegram
- Avito
- сайт avsound.ru
- рекомендации клиентов

## 2. сайт avsound.ru

назначение:
- продажа товаров
- привлечение клиентов в студию
- SEO продвижение
- каталог товаров
- приём заявок

задачи:
- увеличение конверсии
- улучшение карточек товаров
- SEO статьи
- генерация изображений товаров
- повышение среднего чека

## 3. Avito

назначение:
- привлечение клиентов на установку
- продажа товаров
- получение лидов

стратегия:
- регулярная публикация объявлений
- обновление объявлений
- работа с отзывами
- продвижение услуг студии + популярных товаров

## 4. Ozon

назначение: продажа товаров по всей России

задачи:
- создание и улучшение карточек товаров
- SEO оптимизация карточек
- инфографика
- анализ конкурентов
- контроль остатков

## 5. Wildberries

задачи:
- улучшение карточек
- работа с отзывами
- SEO
- управление ассортиментом

## 6. Яндекс Маркет

задачи:
- развитие каталога
- улучшение карточек
- работа с ценами
- повышение рейтинга

## главная цель

объединить все каналы в единую систему:

**студия → сайт → Avito → Ozon → WB → Яндекс Маркет**

чтобы каждый канал усиливал остальные.
