# 🇬🇧 English — второй мозг для языка

Система знаний по английскому Михаила Автономова. Пополняется автоматически: новое слово/фраза из урока, переписки, видео → сразу в нужный файл.

## Профиль обучения
- **Учитель:** Polina Zhuikova (онлайн, Zoom)
- **Уровень:** A1 / Starter–Elementary (beginner)
- **Формат:** индивидуальные занятия ~1–2 раза в неделю + домашка
- **Самостоятельно:** готов уделять **2 часа в день**
- **Старт системы:** 2026-06-24 (импорт переписки с учителем из Telegram)

## Учебники
- **Грамматика Мерфи** (красный, English Grammar in Use) — главная грамматика
- **Speakout** 3rd edition A1 (синий) — основной курс
- **Cambridge Empower Starter** — аудио/видео/доп. материалы
- + рабочая тетрадь, доп. упражнения, словарные таблицы по урокам

## Цель (задана 2026-06-24)
- **Зачем:** путешествия, элементарное общение в дороге.
- **Главное:** **СВОБОДНО ГОВОРИТЬ.** Заказывать еду в кафе/ресторане, в самолёте, в пути; базовый разговор.
- **Приоритет навыка:** **Speaking** №1 (понимание на слух — следом, чтение/письмо — по остаточному).
- **Дедлайна нет** — ровный темп, развиваемся.

## План 2 часа в день — заточен под РАЗГОВОР
1. **20 мин** — слова вслух: повтор [[vocabulary]] + [[phrases]], проговаривать каждое слово громко (не молча!).
2. **40 мин** — РОЛЕВКА: разыгрывать диалоги (кафе/ресторан/самолёт/такси/магазин) из уроков и [[phrases]]. Я за официанта/кассира — ты отвечаешь вслух. Главный блок.
3. **30 мин** — аудирование + shadowing: слушать трек, повторять за диктором как эхо.
4. **30 мин** — минимум грамматики под речь (Present Simple, I'd like, Can I have, How much) — короткие устные подстановки, без письменных простыней.

## Как пополняется
1. Новое слово/фраза из урока/переписки/видео → [[vocabulary]] / [[phrases]]
2. Новый материал → [[materials]]
3. Ошибка на занятии → [[mistakes]]
4. Конец занятия → [[progress]]

> Грамматику Мерфи письменно — по минимуму, как опору для речи. Фокус — рот говорит каждый день. Цель/приоритет заданы выше.
