# Стоимость генерации — Nano Banana (Gemini Image)

Расчёт на основе официального прайса Google AI (Gemini API). Картинка тарифицируется
как **выходные токены**: 1 изображение ≤ 1024×1024 ≈ **1290 выходных токенов**.

## Модель по умолчанию: `gemini-2.5-flash-image`

- Цена выходных токенов: **$30 / 1M токенов**.
- 1 картинка = 1290 ток. → **$0.0387 ≈ $0.039**.
- Входной текст промпта — копейки (~$0.0001). Входное фото для edit (~260–1290 вх. ток.
  по $0.30/1M) → +~$0.0004. Считаем округлённо.

| Объём | Стоимость |
|---|---|
| 1 генерация | **~$0.039** (~3.9 ₽ при 100₽/$) |
| 100 генераций | **~$3.90** |
| 1000 генераций | **~$39** |

## Pro-качество: `gemini-3-pro-image-preview` (Nano Banana Pro)
