# English — система изучения

Папка-мозг для английского. Сюда складываю всё: слова, материалы, ошибки, прогресс.

## Файлы
- [[vocabulary]] — новые слова и фразы (главный файл, пополняется постоянно)
- [[materials]] — что изучали: книги, видео, курсы, статьи, переписки
- [[grammar]] — правила грамматики, которые разбирали
- [[phrases]] — готовые выражения, идиомы, разговорные обороты
- [[mistakes]] — мои частые ошибки → как правильно
- [[progress]] — лог сессий: дата, что прошли, что дальше

## Как работаем
1. Любое новое слово/фраза из переписки, файла, видео → я кладу в `vocabulary.md`
2. Любой материал (ссылка, файл, видео) → фиксирую в `materials.md`
3. Ошибка в моём/твоём тексте → в `mistakes.md`
4. Конец сессии → запись в `progress.md`
