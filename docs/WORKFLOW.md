# Общий алгоритм работы системы

Данный документ описывает полный жизненный цикл обработки одного товара.

Каждая строка Excel является отдельной задачей.

После завершения обработки одного товара система должна переходить к следующему.

# Шаг 1. Импорт Excel

Открыть Excel.

Прочитать все строки.

Для каждой строки определить:

* название;
* бренд;
* модель;
* артикул;
* цену;
* категорию;
* остальные поля.

Если строка пустая —

пропустить.

---

# Шаг 2. Проверка данных

Проверить:

* заполнено ли название;
* есть ли модель;
* есть ли бренд;
* есть ли артикул.

Если информации недостаточно —

отправить товар в отчет.

---

# Шаг 3. Поиск товара

Искать последовательно:

1. Артикул.
2. SKU.
3. Модель.
4. Полное название.
5. Похожее название.

Если найдено несколько совпадений —

не изменять.

Добавить в ручную проверку.

---

# Шаг 4. Если товар найден

Получить Product ID.

Обновить:

* описание;
* характеристики;
* атрибуты;
* изображения.

Не изменять остальные товары.

---

# Шаг 5. Если товар отсутствует

Создать новую карточку.

Заполнить все обязательные поля.

Получить Product ID.

Перейти к следующему этапу.

---

# Шаг 6. Получение характеристик

Найти характеристики.

Использовать только проверенные источники.

Если данные противоречат друг другу —

использовать данные производителя.

---

# Шаг 7. Создание описания

Описание должно:

* быть уникальным;
* профессиональным;
* SEO-оптимизированным;
* полезным.

Не использовать выдуманные характеристики.

---

# Шаг 8. Поиск фотографий

Найти максимально качественные фотографии.

Приоритет:

1. Производитель.
2. Каталог производителя.
3. Дилер.
4. Интернет-магазины.

Цель —

найти три фотографии.

---

# Шаг 9. Проверка фотографий

Проверить:

* совпадает модель;
* совпадает цвет;
* совпадает комплектация;
* нет чужих логотипов;
* нет водяных знаков на товаре;
* нет ценников;
* хорошее качество.

Если фотография плохая —

не использовать.

---

# Шаг 10. Генерация через KIE.ai

Для каждой выбранной фотографии:

* изменить только фон;
* сохранить товар полностью;
* сохранить логотип производителя;
* сохранить форму;
* сохранить цвет;
* сохранить текст.

Получить новое изображение.

---

# Шаг 11. Контроль качества

После генерации проверить:

* качество;
* резкость;
* отсутствие артефактов;
* отсутствие искажений;
* единый стиль каталога.

Если результат плохой —

повторить генерацию.

До трех попыток.

---

# Шаг 12. Загрузка

Загрузить фотографии.

Назначить:

Фото №1

→ главное.

Фото №2

→ галерея.

Фото №3

→ галерея.

---

# Шаг 13. Удаление старых фотографий

После успешной загрузки новых фотографий:

Удалить старые фотографии только этого товара.

Не затрагивать фотографии других товаров.

---

# Шаг 14. Финальная проверка карточки

Проверить:

✓ название;

✓ описание;

✓ характеристики;

✓ категория;

✓ цена;

✓ главное фото;

✓ галерея;

✓ отсутствие битых изображений.

Если проверка не пройдена —

добавить товар в отчет.

---

# Шаг 15. Отчет

Добавить запись:

* Product ID;
* название;
* статус;
* создан;
* обновлен;
* фотографии;
* ошибки;
* предупреждения.

---

# Завершение

После завершения обработки текущего товара перейти к следующей строке Excel.

После обработки всех строк сформировать итоговый отчет.

Никогда не прерывать обработку всего файла из-за ошибки одного товара.
