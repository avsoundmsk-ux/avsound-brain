# 💬 Phrases — разговорные фразы и выражения

Из уроков Polina (A1 Starter). Кафе/магазин/вежливость.

## Вежливость и реакции
| Фраза | Значение |
|---|---|
| Nice to meet you. | Приятно познакомиться. |
| See you later. | Увидимся позже. |
| Of course! / Sure! | Конечно! |
| No problem. | Без проблем. |
| Lovely! | Прелестно! |
| That's right! | Точно! / Правильно! |
| Really? | Правда? |
| Actually, ... | Вообще-то, ... |
| I suppose. | Я полагаю. |
| I think ... | Я думаю, ... |
| Good morning / afternoon / evening. | Доброе утро / день / вечер. |

## В кафе / магазине
| Фраза | Значение |
|---|---|
| I'd like a cup of tea, please. | Я хотел бы чашку чая, пожалуйста. |
| Can I have a piece of cake? | Можно мне кусочек торта? |
| Anything else? | Что-нибудь ещё? |
| Here you are. / There you go. | Вот, пожалуйста / держите. |
| Enjoy your meal! | Приятного аппетита! |
| Excuse me. | Извините. |
| How much is it? | Сколько стоит? |

## Small talk
| Фраза | Значение |
|---|---|
| How are things? | Как дела? |
| How are you? | Как ты? |
| Good things / bad things | Хорошие / плохие вещи |
| Things to do | Дела |

# ✈️ Travel survival — выживание в дороге (цель Михаила)

## Заказ еды (кафе/ресторан) — приоритет №1
| Фраза | Значение |
|---|---|
| A table for two, please. | Столик на двоих, пожалуйста. |
| Can I see the menu, please? | Можно меню, пожалуйста? |
| I'd like a coffee, please. | Я хотел бы кофе, пожалуйста. |
| Can I have the chicken? | Можно мне курицу? |
| What do you recommend? | Что посоветуете? |
| Is it spicy? | Это острое? |
| No onions, please. | Без лука, пожалуйста. |
| Still or sparkling? | Без газа или с газом? |
| The bill, please. / Check, please. | Счёт, пожалуйста. |
| Can I pay by card? | Можно оплатить картой? |
| Keep the change. | Сдачи не надо. |

## Самолёт / аэропорт
| Фраза | Значение |
|---|---|
| Where is the gate? | Где выход на посадку? |
| Is this seat free? | Это место свободно? |
| Can I have some water, please? | Можно воды, пожалуйста? |
| Chicken or fish? | Курица или рыба? (бортпроводник) |
| Can I have a blanket? | Можно плед? |
| Where is my baggage? | Где мой багаж? |
| I have a connecting flight. | У меня стыковочный рейс. |

## Такси / в пути / дорога
| Фраза | Значение |
|---|---|
| How much is it to the centre? | Сколько до центра? |
| Take me to this address, please. | Отвезите по этому адресу. |
| Stop here, please. | Остановите здесь. |
| How do I get to ...? | Как мне добраться до ...? |
| Is it far? | Это далеко? |
| Turn left / right. | Поверните налево / направо. |
| Where is the toilet? | Где туалет? |

## Отель
| Фраза | Значение |
|---|---|
| I have a reservation. | У меня бронь. |
| A room for one night. | Номер на одну ночь. |
| What time is check-out? | Во сколько выезд? |
| The Wi-Fi password, please? | Пароль от Wi-Fi? |

| Can you repeat, please? | Повторите, пожалуйста? |
| Can you speak slowly, please? | Говорите медленнее, пожалуйста. |
| How do you say ... in English? | Как сказать ... по-английски? |
