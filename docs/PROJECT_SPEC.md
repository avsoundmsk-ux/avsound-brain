# PROJECT_SPEC.md

# Общая цель проекта

Создать полностью автоматизированную систему наполнения и обновления каталога интернет-магазина AV-Sound.ru.

Система должна максимально сократить ручную работу и обеспечить единый высокий стандарт качества для всех карточек товаров.

---

# Основные задачи

Система должна автоматически:

* читать Excel-файл с товарами;
* находить существующие товары на сайте;
* создавать отсутствующие товары;
* обновлять существующие товары;
* автоматически искать достоверные характеристики;
* автоматически создавать качественные описания;
* автоматически искать лучшие фотографии;
* обрабатывать фотографии через KIE.ai;
* загружать фотографии в WooCommerce;
* создавать подробный отчет.

---

# Источник данных

Единственным источником товаров является Excel-файл.

Каждая строка Excel представляет один товар.

Каждая строка должна быть обработана независимо от остальных.

---

# Алгоритм обработки товара

Для каждой строки Excel необходимо выполнить следующий алгоритм:

1. Прочитать строку.
2. Определить артикул.
3. Определить бренд.
4. Определить модель.
5. Найти товар на сайте.
6. Если найден — обновить.
7. Если не найден — создать.
8. Найти характеристики.
9. Создать описание.
10. Найти изображения.
11. Проверить изображения.
12. Обработать изображения через KIE.ai.
13. Загрузить изображения.
14. Удалить старые фотографии только у этого товара.
15. Добавить информацию в отчет.

После завершения перейти к следующей строке Excel.

---

# Поиск товара

Поиск выполнять в следующем порядке:

1. По артикулу.
2. По точной модели.
3. По точному названию.
4. По похожему названию.

Приоритет всегда отдавать артикулу.

Если совпадение вызывает сомнения — не изменять товар.

---

# Создание товара

Если товар отсутствует:

Создать новую карточку.

Заполнить:

* название;
* бренд;
* категория;
* цена;
* описание;
* краткое описание;
* характеристики;
* атрибуты;
* SEO-заголовок;
* SEO-описание.

---

# Обновление товара

Если товар найден:

Обновлять только недостающие или устаревшие данные.

Не удалять корректную информацию без необходимости.

Не изменять данные, отсутствующие в Excel, если они уже заполнены корректно.

---

# Характеристики

Все характеристики искать автоматически.

Приоритет источников:

1. Производитель.
2. Официальный каталог.
3. Официальный дилер.
4. Надежные магазины автозвука.

Если данные противоречат друг другу — использовать источник производителя.

Если информации нет — добавить товар в отчет для ручной проверки.

---

# Описание товара

Описание должно:

* быть уникальным;
* быть профессиональным;
* быть полезным;
* быть SEO-оптимизированным;
* не содержать воды;
* не содержать ложной информации.

---

# Работа с фотографиями

Для каждого товара необходимо стремиться получить три качественные фотографии.

Если доступно больше трех — автоматически выбрать лучшие.

Если доступно меньше трех — использовать максимально возможное количество.

Каждая фотография должна пройти проверку качества перед использованием.

---

# Генерация изображений

Все фотографии должны пройти обработку через KIE.ai.

Разрешается изменять только фон.

Товар должен остаться абсолютно идентичным оригиналу.

После генерации изображения должны выглядеть как единый каталог интернет-магазина.

---

# WooCommerce

После завершения обработки:

* загрузить фотографии;
* назначить главное изображение;
* заполнить галерею;
* обновить карточку товара.

---

# Масштабируемость

Архитектура должна позволять легко добавлять:

* новые поставщики данных;
* новые модели AI;
* новые генераторы изображений;
* новые источники фотографий;
* новые маркетплейсы;
* новые форматы отчетов.

Проект должен быть модульным.

Новые функции должны добавляться без переписывания существующего кода.

---

# Основной принцип разработки

Каждая функция должна выполнять только одну задачу.

Код должен быть максимально читаемым.

Любое потенциально опасное действие должно иметь предварительную проверку.

Любая ошибка должна логироваться.

После любого запуска пользователь должен получать подробный отчет.

---

# Конечная цель

После обработки Excel-файла каталог AV-Sound.ru должен автоматически получить:

* новые товары;
* обновленные характеристики;
* качественные описания;
* профессиональные фотографии;
* единый фирменный стиль;
* готовые к продаже карточки товаров.

При этом ни один товар вне Excel не должен быть изменен.
