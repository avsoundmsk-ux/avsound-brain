# CATEGORY_RULES.md

Правила назначения категории WooCommerce товару из Excel.

# Принципы (утверждено)
- НЕ создавать новые категории автоматически.
- Категория из Excel не сопоставлена → товар НЕ создавать → ручная проверка.
- Несколько похожих категорий → ручная проверка.
- Категория назначается ТОЛЬКО по утверждённой таблице соответствия (ниже).
- При обновлении товара категорию НЕ менять, если текущая корректна.
- При создании товара категория обязательна.

# УТВЕРЖДАЕМАЯ ТАБЛИЦА СООТВЕТСТВИЯ

| Excel category | WooCommerce name | category_id | confidence | comment |
|---|---|---|---|---|
| Усилители | Усилители | **256** | HIGH | верхний уровень; есть дубль 312 (дочерний) — утвердить, что пишем в 256 |
| Магнитолы | Магнитолы | **320** | HIGH | верхний; дубль 321 (дочерний) — утвердить 320 |
| Сабвуфер / «Сабвуфер » | Сабвуферы | **294** | HIGH | верхний; дубль 310 (дочерний) — утвердить 294; пробел в Excel нормализуем |

| Автоакустика | Акустика | **252** | HIGH | УТВЕРЖДЕНО: всю «Автоакустику» заливать в «Акустика»(252). Сайтовую «Автоакустика»(355) НЕ использовать |
| Камеры | — | — | **LOW → ручная проверка** | УТВЕРЖДЕНО: точной категории нет → ручная проверка |
| «категория» | — | — | — | артефакт строки-заголовка, игнорировать |

# Решение по confidence
- **HIGH** → назначать автоматически по таблице.
- **LOW / нет в таблице / несколько кандидатов** → НЕ назначать, товар в ручную проверку.

# Утверждённые решения (финал)
1. Дубли имён → пишем в **верхний уровень**: Усилители **256**, Сабвуферы **294**, Магнитолы **320**. ✅
2. «Автоакустика» (71 товар) → всё в **«Акустика» (252)**. ✅
3. «Камеры» (1) → **ручная проверка**. ✅
