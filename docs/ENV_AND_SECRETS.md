# ENV_AND_SECRETS.md

> Заготовка. Заполняем вместе.

# Назначение

Список секретов и переменных окружения. Ключи только в .env, НИКОГДА в коде/гите.

# Переменные (.env)
| Переменная | Сервис | Назначение |
|---|---|---|
| WOO_CK | WooCommerce | Consumer Key |
| WOO_CS | WooCommerce | Consumer Secret |
| KIE_API_KEY | KIE.ai | генерация изображений |
| BRAVE_KEY | Brave Search | поиск фото/данных |
| (опц.) OPENROUTER_KEY | LLM | описания/характеристики |

# Правила
- `.env` в `.gitignore`.
- Ротация ключей при утечке (см. инцидент в памяти).
- Не логировать значения ключей.
- В коде брать через config_manager / os.environ, без хардкода.
