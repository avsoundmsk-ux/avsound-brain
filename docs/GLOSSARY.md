# GLOSSARY.md

Единые термины проекта, чтобы документы и код понимались однозначно.

| Термин | Определение |
|---|---|
| Товар | одна позиция каталога = одна строка Excel (колонка E «название») |
| Бренд | производитель (Pride, JBL, Nakamichi, Ural…). В Excel — из секции-заголовка/названия. На сайте — атрибут «Производитель» |
| Модель | конкретное наименование изделия без бренда и слова-типа (напр. `Uno plus`, `M1-216`) |
| Артикул | заводской код производителя. **В этом Excel ОТСУТСТВУЕТ** |
| SKU | код товара в WooCommerce. Генерируется сайтом (`BRAND-TYPE-NAME-NNNN-HASH`). В Excel отсутствует |
| Product ID | внутренний числовой id WooCommerce (напр. 41560) |
| Категория (Excel) | колонка D (напр. «Автоакустика») |
| Категория (сайт) | узел дерева WooCommerce по id (см. CATEGORY_RULES.md) |
| Подкатегория | конечный узел дерева (Компонентная/Коаксиальная/…); определяется анализом, не только Excel |
| Различающий токен | признак варианта: импеданс (D2/D4), размер (10"/12"/16см/6x9), версия (ver.2/mkii) |
| **Вариант** | товар, отличающийся различающим токеном. **Считается ОТДЕЛЬНЫМ товаром** (D2 ≠ D4, 10" ≠ 12"). Совпадение засчитывается только при совпадении токена |
| Комплектация | состав поставки; влияет на выбор фото (не использовать фото другой комплектации) |
| Уверенное совпадение | score ≥ 90 и единственный кандидат (MATCHING_RULES.md) |
| Ручная проверка | статус товара, который нельзя менять автоматически (сомнение/конфликт/нет данных) |
| Dry Run | прогон без изменений сайта (режим по умолчанию) |
| Live Run | прогон с реальными изменениями; только после подтверждения пользователя |
| Hero / главное фото | первое изображение карточки (`images[0]`) |
| Галерея | остальные изображения карточки (`images[1..]`) |
| norm() | нормализованная строка названия/модели для сравнения (MATCHING_RULES.md) |
| row_key | ключ строки Excel = хэш(норм. название + бренд + категория); идентификатор для State Manager |
| Аудит | полная запись «было/стало» по каждому изменению (audit_manager) |
| Бэкап | сохранённое прошлое состояние товара/фото перед изменением (для отката) |
