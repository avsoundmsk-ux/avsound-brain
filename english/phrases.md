# 💬 Phrases — разговорные фразы и выражения

Из уроков Polina (A1 Starter). Кафе/магазин/вежливость.

## Вежливость и реакции
| Фраза | Значение |
|---|---|
| Nice to meet you. | Приятно познакомиться. |
| See you later. | Увидимся позже. |
| Of course! / Sure! | Конечно! |
| No problem. | Без проблем. |
| Lovely! | Прелестно! |
| That's right! | Точно! / Правильно! |
| Really? | Правда? |
| Actually, ... | Вообще-то, ... |
| I suppose. | Я полагаю. |
| I think ... | Я думаю, ... |
| Good morning / afternoon / evening. | Доброе утро / день / вечер. |

## В кафе / магазине
| Фраза | Значение |
|---|---|
| I'd like a cup of tea, please. | Я хотел бы чашку чая, пожалуйста. |
| Can I have a piece of cake? | Можно мне кусочек торта? |
| Anything else? | Что-нибудь ещё? |
| Here you are. / There you go. | Вот, пожалуйста / держите. |
| Enjoy your meal! | Приятного аппетита! |
| Excuse me. | Извините. |
| How much is it? | Сколько стоит? |

## Small talk
| Фраза | Значение |
|---|---|
| How are things? | Как дела? |
| How are you? | Как ты? |
| Good things / bad things | Хорошие / плохие вещи |
| Things to do | Дела |
