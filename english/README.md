# 🇬🇧 English — второй мозг для языка

Система знаний по английскому Михаила Автономова. Пополняется автоматически: новое слово/фраза из урока, переписки, видео → сразу в нужный файл.

## Профиль обучения
- **Учитель:** Polina Zhuikova (онлайн, Zoom)
- **Уровень:** A1 / Starter–Elementary (beginner)
- **Формат:** индивидуальные занятия ~1–2 раза в неделю + домашка
- **Самостоятельно:** готов уделять **2 часа в день**
- **Старт системы:** 2026-06-24 (импорт переписки с учителем из Telegram)

## Учебники
- **Грамматика Мерфи** (красный, English Grammar in Use) — главная грамматика
- **Speakout** 3rd edition A1 (синий) — основной курс
- **Cambridge Empower Starter** — аудио/видео/доп. материалы
- + рабочая тетрадь, доп. упражнения, словарные таблицы по урокам

## Файлы системы
| Файл | Что внутри |
|---|---|
| [[vocabulary]] | мастер-словарь по категориям (~300 слов) — главный |
| [[grammar]] | пройденные грамматические темы + правила |
| [[phrases]] | разговорные фразы и выражения |
| [[homework_log]] | хронология домашек от учителя |
| [[mistakes]] | мои ошибки → как правильно (пополнять на занятиях) |
| [[progress]] | лог занятий и прогресс |
| [[materials]] | книги/аудио/видео + где лежат файлы |
| `source/` | исходники: словарь.docx, экспорт чата, дамп уроков |

## План 2 часа в день (черновик)
1. **30 мин** — повтор слов из [[vocabulary]] (интервальное повторение)
2. **30 мин** — грамматика Мерфи + упражнения [[grammar]]
3. **30 мин** — Speakout/Empower: чтение текста, аудирование (треки)
4. **30 мин** — перевод предложений + говорение вслух (диалоги из уроков)

## Как пополняется
1. Новое слово/фраза из урока/переписки/видео → [[vocabulary]] / [[phrases]]
2. Новый материал → [[materials]]
3. Ошибка на занятии → [[mistakes]]
4. Конец занятия → [[progress]]

> **Уточнить у Михаила:** цель языка (путешествия / работа / клиенты?), дедлайн, какой навык важнее (speaking / понимание на слух / чтение).
