# Назначение

Единые термины проекта, чтобы документы и код понимались однозначно.

| Термин | Определение |
|---|---|
| Товар | одна позиция каталога = одна строка Excel |
| Артикул | заводской код производителя (из Excel) |
| SKU | код товара в WooCommerce |
| Модель | конкретное наименование изделия (напр. Uno plus) |
| Бренд | производитель (напр. Pride) |
| Вариант | TODO (цвет/импеданс/размер — считается отдельным товаром?) |
| Комплектация | состав поставки (влияет на совпадение фото) |
| Product ID | внутренний id WooCommerce |
| Уверенное совпадение | см. MATCHING_RULES.md |
| Dry Run | прогон без изменений сайта |
| Live Run | прогон с реальными изменениями (только после подтверждения) |
| Hero / главное фото | первое изображение карточки |
