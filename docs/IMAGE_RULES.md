# IMAGE_RULES.md

# Правила обработки изображений

Данный документ определяет единый стандарт изображений для каталога AV-Sound.ru.

Каждая карточка товара должна выглядеть так, словно все фотографии были сделаны в одной профессиональной фотостудии.

---

# Основная цель

Получить максимально качественные, одинаковые по стилю фотографии товаров.

После обработки пользователь не должен понимать, что фотографии были собраны из разных источников.

Весь каталог должен выглядеть единым.

---

# Поиск изображений

Для каждого товара необходимо стремиться найти **3 качественные фотографии**.

Минимум — одна фотография.

Максимум — три фотографии.

Если найдено больше трех —

автоматически выбрать лучшие.

---

# Приоритет источников

Искать изображения строго в следующем порядке:

1. Официальный сайт производителя.
2. Официальный каталог производителя.
3. Официальный дилер.
4. Авторизованные магазины.
5. Крупные интернет-магазины.
6. Другие источники только при отсутствии предыдущих.

---

# Требования к изображениям

Изображение должно:

* соответствовать модели;
* соответствовать бренду;
* соответствовать цвету;
* соответствовать комплектации;
* иметь высокое разрешение;
* быть четким;
* быть без сильного JPEG-сжатия.

---

# Запрещенные изображения

Не использовать изображения:

* другого товара;
* другой модели;
* другого цвета;
* другой комплектации;
* с плохим качеством;
* размытые;
* шумные;
* обрезанные;
* с сильными отражениями;
* со скриншотов.

---

# Чужие логотипы

Если чужой логотип находится:

* на самом товаре;
* на коробке;
* на экране;
* на корпусе;
* на динамике;
* на упаковке,

использовать изображение запрещено.

Если чужой логотип расположен:

* на фоне;
* в углу изображения;
* на белом поле;
* вне товара,

изображение использовать разрешается, если после генерации через KIE.ai логотип полностью исчезнет.

---

# Водяные знаки

Если водяной знак находится на товаре —

изображение запрещено.

Если водяной знак находится только на фоне —

разрешено использовать.

После генерации не должно остаться следов водяного знака.

---

# Количество фотографий

Если найдено:

1 фотография —

использовать одну.

2 фотографии —

использовать обе.

3 фотографии —

использовать все три.

Больше трех —

выбрать лучшие три.

Использование только одной фотографии при наличии трех считается ошибкой алгоритма.

---

# Интеллектуальный выбор лучших фотографий

Если найдено больше трех фотографий —

оценить каждую.

Критерии оценки:

* качество;
* разрешение;
* резкость;
* отсутствие шума;
* отсутствие водяных знаков;
* отсутствие текста;
* отсутствие ценников;
* официальный источник;
* совпадение модели;
* совпадение цвета;
* совпадение комплектации.

Оставить три фотографии с максимальным рейтингом.

---

# Разнообразие фотографий

По возможности выбрать:

Фото №1

главный фронтальный вид.

Фото №2

другой ракурс.

Фото №3

разъемы, задняя часть, комплектность или упаковка.

Не использовать три одинаковых фотографии.

---

# Генерация через KIE.ai

Каждая фотография должна пройти обработку.

Разрешается изменить только фон.

Запрещается изменять:

* товар;
* форму;
* размеры;
* логотип производителя;
* текст;
* цвет;
* кнопки;
* разъемы;
* материалы;
* отражения товара.

---

# Стиль каталога

Все фотографии должны иметь единый стиль.

Фон:

* темный графит;
* черный;
* студийное освещение;
* мягкий свет;
* легкое свечение за товаром;
* легкая тень.

Без:

* людей;
* автомобилей;
* мебели;
* предметов;
* текста;
* рекламы.

---

# Размер

Все изображения:

* квадратные;
* одинакового размера;
* одинакового качества.

Рекомендуемый размер:

1600×1600.

---

# Контроль качества

После генерации автоматически проверить:

✓ модель совпадает;

✓ логотип производителя сохранен;

✓ текст читается;

✓ товар не изменился;

✓ нет артефактов;

✓ нет водяных знаков;

✓ нет чужих логотипов;

✓ нет лишних объектов;

✓ фон соответствует фирменному стилю;

✓ изображение четкое.

Если хотя бы один пункт не выполнен —

повторить генерацию.

Максимум три попытки.

---

# Финальная загрузка

После успешной проверки:

Фото №1

→ главное изображение.

Фото №2

→ галерея.

Фото №3

→ галерея.

После успешной загрузки разрешается удалить старые фотографии только данного товара.

---

# Главный принцип

Лучше использовать одну идеальную фотографию, чем три плохих.

Но если доступны три качественные фотографии — использовать все три.
