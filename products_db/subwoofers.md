# сабвуферы

## типы
- пассивные (нужен внешний усилитель)
- активные (встроенный усилитель)
- для SQ (качество баса)
- для SPL (громкость баса)

## размеры
- 8" — компактный
- 10" — универсальный
- 12" — стандарт
- 15"+ — SPL

## типы корпусов
- закрытый ящик (ЗЯ) — точный бас
- фазоинвертор (ФИ) — громче, глубже
- бандпасс — очень громко
